# 📖 Executive Summary

The **MUST Smart Campus Navigation & Student Assistance Platform (M-SNAP)** is a modern Smart Campus solution designed to improve navigation and accessibility within **Mbeya University of Science and Technology (MUST)**.

The platform enables students, staff, visitors, parents, and guests to quickly locate buildings, lecture halls, laboratories, administrative offices, hostels, cafeterias, libraries, health centers, and other important university facilities.

Unlike a traditional university website, M-SNAP focuses on solving one of the most common daily problems faced by students—finding locations within the university campus.

Using an interactive digital map, intelligent search, and navigation technologies, users can reach their destinations faster while reducing confusion and saving time.

The project is initially developed for MUST but is designed to be scalable for deployment to universities across Tanzania.

---

# 🌍 Background

Every academic year, thousands of students join universities across Tanzania.

One of the biggest challenges faced by first-year students is understanding the university environment.

Students often spend several days trying to locate:

- Lecture halls
- Laboratories
- Departments
- Administrative offices
- Library
- Student hostels
- Cafeterias
- ICT Office
- Finance Office
- Health Centre

Visitors and parents experience the same challenge when visiting the university.

Although universities have official websites, these websites mainly provide academic information and announcements rather than helping users navigate the physical campus.

As a result, students waste valuable academic time asking for directions or walking around searching for buildings.

This project aims to solve this problem.

# 🚀 Getting Started

This section explains how to set up and run the project locally.

## System Requirements

Before running the project, ensure you have the following installed:

- Python 3.12+
- PostgreSQL 16+
- Git
- Docker (Optional)
- Docker Compose (Optional)
- Node.js 20+ (Frontend)
- npm or yarn
- VS Code (Recommended)

---

# Clone Repository

```bash
git clone https://github.com/mtundudev/must-smart-campus-guide.git
```

Move into the project directory.

```bash
cd must-smart-campus-guide
```

---

# Create Virtual Environment

Windows

```bash
python -m venv .venv
```

Linux / macOS

```bash
python3 -m venv .venv
```

Activate the virtual environment.

Windows

```bash
.venv\Scripts\activate
```

Linux / macOS

```bash
source .venv/bin/activate
```

---

# Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Configure Environment Variables

Create a `.env` file.

Example:

```env
APP_NAME=MUST Smart Campus Navigation

APP_ENV=development

SECRET_KEY=CHANGE_THIS_SECRET_KEY

ALGORITHM=HS256

ACCESS_TOKEN_EXPIRE_MINUTES=30

DATABASE_URL=postgresql://username:password@localhost:5432/must_navigation

HOST=127.0.0.1

PORT=8000
```

---

# Database Migration

Run Alembic migrations.

```bash
alembic upgrade head
```

---

# Run Development Server

```bash
uvicorn app.main:app --reload
```

API will be available at

```
http://127.0.0.1:8000
```

Swagger Documentation

```
http://127.0.0.1:8000/docs
```

ReDoc Documentation

```
http://127.0.0.1:8000/redoc
```

---

# Docker Installation

Build the Docker image.

```bash
docker compose build
```

Start all containers.

```bash
docker compose up -d
```

Stop containers.

```bash
docker compose down
```
